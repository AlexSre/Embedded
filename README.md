# Embedded Projects

This repository contains a collection of embedded systems projects developed for learning, experimentation, and preparation for robotics and microcontroller-based applications.

The projects focus on STM32 development, UART communication, master-slave architectures, packet-based protocols, real-time control concepts, motor-control abstraction, and future autonomous robotics applications.

## Purpose

The goal of this repository is to document and organize my embedded systems learning path through practical projects. Each project is designed to demonstrate a specific concept used in microcontroller programming and autonomous robotics.

Main objectives:

- learn embedded C programming;
- configure STM32 peripherals using STM32CubeMX and STM32CubeIDE;
- work with GPIO, UART, timers, PWM, and communication protocols;
- build simple but expandable embedded architectures;
- create reusable code for robotics-related applications;
- prepare for more advanced autonomous vehicle and NXP Cup-style projects.

## Hardware Used

Current hardware used across the projects:

- 2 × STM32 Nucleo-F446RE development boards;
- Nexys A7-50T FPGA board;
- Plusivo Uno R3 starter kit;
- jumper wires;
- USB cables;
- external modules and sensors for future extensions.

The main development board used at this stage is the STM32 Nucleo-F446RE, based on the STM32F446RE microcontroller with an ARM Cortex-M4 core.

## Software and Tools

The projects are developed using:

- STM32CubeIDE;
- STM32CubeMX;
- STM32CubeProgrammer;
- STM32 HAL drivers;
- Embedded C;
- Git and GitHub.

## Learning Goals

Through these projects, I aim to improve my understanding of:

- embedded C;
- microcontroller peripherals;
- UART communication;
- GPIO control;
- real-time embedded logic;
- protocol design;
- hardware debugging;
- clean technical documentation.
